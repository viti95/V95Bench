#############
# V95 Bench #
#############

 Benchmark designed to test performance of all kind of graphics cards 
 in DOS environments.

#########################
# Supported video cards #
#########################

 - MDA
 - Hercules
 - CGA
 - EGA
 - MCGA
 - VGA
 - ATI Small Wonder series
 - Plantronics ColorPlus

###############
# Executables #
###############

 - V95_16.EXE: 16-bit executable, runs on 8088 / 8086 processors onwards.
               Recommended for processors up to 286.
 - V95_32.EXE: 32-bit proctected mode executable, runs on 386 processors
               onwards. Recommended for 386 processors onwards.

######################
# Build instructions #
######################

 1) Install Open Watcom C version 2.0
 2) Run clean.bat
